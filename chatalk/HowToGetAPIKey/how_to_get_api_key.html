How to Get API Key?¶

Sign Up¶

If you do not have Chatalk API key, click How to Get API Key? button to signup.

A small window will appear where you can enter your name and email. Please fill the form and click Send Email to proceed.

UI label

Description

User Name

your nickname

User Email

Email to receive API Key

Send Email

Receive verification email

Privacy Policy

Open Chatalk privacy policy document.

If we send verification email to your address successfully, a dialog box below will appear.

Get API Key¶

Finally, you can receive API key using verification link clicking here.

Please write your API key and click Save button to use Chatalk.

If you have any other questions, read FAQ.
